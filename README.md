# AppointMe

AppointMe is a full-stack appointment scheduling platform that allows users to book, manage, and review appointments across multiple service categories.

## Features

- User Authentication (Client, Provider, Admin)
- Real-time Appointment Booking
- Provider Availability Management
- Secure Role-Based Access Control
- Appointment Payments
- Ratings & Reviews System
- Responsive UI

## Tech Stack

- React
- TypeScript
- Tailwind CSS
- Supabase
- PostgreSQL
- Vite
- shadcn/ui
- React Query

## User Roles

### Client
- Browse Services
- Book Appointments
- Make Payments
- Leave Reviews

### Provider
- Manage Profile
- Set Availability
- Confirm Bookings
- View Dashboard

### Admin
- Manage Categories
- Manage Users & Roles
- Monitor Appointments
- Platform Moderation

## Database Tables

- profiles
- user_roles
- categories
- services
- availability
- appointments
- payments
- reviews

## Project Highlights

- Secure authentication with Supabase Auth
- Row Level Security (RLS) implementation
- Complete booking lifecycle: Browse → Book → Pay → Review
- Modern and responsive user interface
- Scalable architecture for future enhancements
